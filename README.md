# 🚀 Project Shravan  
### Smart Assistive System using Computer Vision & Embedded Systems  

---

## 📌 Overview  
**Project Shravan** is an intelligent assistive system that integrates **Computer Vision, Machine Learning, and Embedded Systems** to create a real-time interactive solution.

The system captures user gestures through a camera, processes them using AI models, and communicates with microcontrollers like Arduino/ESP32 to perform physical actions.

---

## 🎯 Features  

- 🧠 Real-time Gesture Recognition  
- ⚡ Fast and Efficient Processing  
- 🔌 Hardware Integration (Arduino / ESP32)  
- 📡 Serial/Wireless Communication  
- 🧩 Modular Code Structure  
- 📈 Scalable Architecture
